## What is the Quake III Arena source release?

Quake III Arena is a first-person shooter released by id Software in 1999. In 2005, id Software released the full source code under the GNU General Public License v2. This documentation covers the GPL source release, explaining the architecture, systems, and tools that make up the engine.

The codebase is written primarily in C, with a modular architecture that separates the engine core from game logic via a QVM (Quake Virtual Machine) bytecode interface. This design allows mods to replace game logic — weapons, entities, AI, UI — without recompiling the engine itself.

## Key features

- **QVM sandboxing**: Game, cgame, and UI modules run as bytecode inside the QVM virtual machine, enabling mods without engine recompilation
- **BSP rendering**: The renderer uses Binary Space Partitioning trees for efficient occlusion and rendering of complex indoor environments
- **Shader system**: A text-based shader scripting language controls surface appearance, animation, blending, and special effects
- **Delta-compressed networking**: The server sends only the *difference* between game states each frame, minimizing bandwidth over UDP
- **AAS pathfinding**: Bots navigate using an Area Awareness System — a precomputed reachability graph that allows sophisticated movement decisions
- **Cross-platform**: Builds on Windows (MSVC), GNU/Linux (GCC), and macOS
